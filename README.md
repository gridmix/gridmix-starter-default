# Default starter for Gridmix

This is the project you get when you run `gridmix create new-project`.

### 1. Install Gridmix CLI tool if you don't have

`npm install --global @gridmix/cli`

### 2. Create a Gridmix project

1. `gridmix create my-gridmix-site` to install default starter
2. `cd my-gridmix-site` to open the folder
3. `gridmix develop` to start a local dev server at `http://localhost:8080`
4. Happy coding 🎉🙌
